# Vulnerability Scanner

A simple Python-based vulnerability scanner developed for educational and internship purposes.

## Features

- Scan common ports
- Detect open services
- Grab service banners
- Identify potentially weak configurations
- Generate vulnerability reports

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python scanner.py
```

Enter a target IP address or domain when prompted.

## Example

```bash
Enter IP address or domain: scanme.nmap.org
```

## Output

A vulnerability report is generated in the `reports` folder.

## Disclaimer

This tool is intended for educational purposes and should only be used on systems you own or have permission to test.
